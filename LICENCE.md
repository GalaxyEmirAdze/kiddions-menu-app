## 🧩 About / What Is This Tool?

**Kiddions Mod Menu** (also known as **Modest Menu**) is an **external** mod menu for GTA 5. Unlike internal mods that inject code into the game, Kiddions runs as a separate process and interacts with the game externally [citation:2][citation:6]. This architecture makes it more stable and reduces the risk of detection compared to internal injectors.

It is widely used and trusted in the modding community for being free, regularly updated, and offering a wide range of features [citation:11].

## ⚙️ Configuration

Settings are stored in `config.json` at `%APPDATA%\KiddionsModMenu\config.json`.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `menu_hotkey` | String | `"F5"` | Key to toggle menu |
| `auto_attach` | Boolean | `true` | Auto-attach on startup |
| `process_name` | String | `"GTA5.exe"` | Target process |
| `safe_mode` | Boolean | `true` | Restricts unstable features |
| `log_level` | String | `"info"` | `"debug"`, `"info"`, `"error"` |

**Is this detectable?**

**Does it work on GTA Online?**
Yes, but use with caution. Rockstar actively bans players who use mods in online mode. It is recommended to use only in invite-only or private sessions [citation:1][citation:5].

**Is this malware?**
No — but antivirus may flag it as a false positive. Download only from the official source.

**Does the tool need updates after game patches?**
Yes. Rockstar updates may change offsets. The tool must be updated to match the current game version [citation:11].

**What is Modest Menu?**
